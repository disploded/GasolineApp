This app was created by Gasoline, a team of year 10 students from Lytton High.
This app is made specifically for 1920x1080 monitors and Windows 10 computers.
It works fine on monitors with higher resolutions, but anything lower will have a bad experience.

To use the app:
You shouldn't modify anything inside of the folder, otherwise the app will break and won't be able to run
Run the .exe file to start the app
Navigate to the "Encrypt" page
After encrypting your file(s), you can find them in the folder under the "Encryptions"

Instructions on how to use the encrypter can be found inside of the app's encryption page
Decrypting the folders is done by simplying pressing on the .zip folder and entering your password-protected key

Thank you for using our app!
